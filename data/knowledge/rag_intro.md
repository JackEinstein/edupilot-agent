# RAG 学习笔记

RAG 是 Retrieval-Augmented Generation 的缩写，中文通常叫检索增强生成。

普通大模型回答问题时，主要依赖模型参数中的知识，可能会出现过时、幻觉或无法引用具体资料的问题。

RAG 的核心思想是：先从外部知识库中检索相关资料，再把检索到的内容和用户问题一起交给大模型生成答案。

一个典型 RAG 流程包括：文档加载、文本切分、向量化、向量数据库存储、相似度检索、生成回答。

文本切分的作用是把长文档拆成较小片段，方便后续检索。

向量化的作用是把文本转换成向量表示，使系统可以根据语义相似度找到相关内容。

向量数据库用于保存文本片段和对应向量。

在 EduPilot Agent 中，RAG 可以用来检索课程资料、项目说明、学习笔记，然后让 Tutor 节点基于资料讲解，而不是只靠模型自由发挥。
